# WhatsApp Daily Reports — نظام مستقل

تطبيق منفصل لإرسال سؤال يومي للموظفين عبر WhatsApp (Green API) وجمع الردود وإرسال تقرير للمدير.

## الصفحات (لوحة شاملة)

| القسم | المحتوى |
|-------|---------|
| **الرئيسية** | إحصائيات · حالة النظام · ملخص اليوم · إرسال فوري |
| **الموظفون** | إضافة · تعديل · حذف · بحث · إرسال سؤال فردي |
| **التقارير** | أرشيف التواريخ · ردود الموظفين · معاينة وإرسال تقرير المدير |
| **الإعدادات** | جدولة مكة (12 ساعة) · الرسائل · Green API · Webhook |

## التشغيل المحلي

```bash
cd whatsapp
copy config.example.json config.json   # Windows
# عدّل config.json — Green API

start.bat
# أو: python server.py
```

يفتح على: **http://localhost:8090**

**الإنتاج (Render):** https://whatsapp1-7sdw.onrender.com

## الرفع على Render

1. ارفع مجلد `whatsapp/` كمستودع أو subdirectory
2. استخدم `render.yaml` المرفق
3. بعد النشر: افتح `/` (لوحة التحكم)
4. أضف Green API + رابط السيرفر العام
5. اضغط **ضبط Webhook**

Webhook: `https://whatsapp1-7sdw.onrender.com/api/webhook/green`

## الجدولة (توقيت مكة المكرمة — 12 ساعة)

- **الجدولة التلقائية:** فعّلها وحدد الوقت بص/م والأيام
- **الإرسال الفوري:** أزرار «إرسال فوري» تعمل حتى لو الجدولة معطّلة
- **افتراضي:** سؤال 7:00 م · تقرير 12:00 ص (منتصف الليل)
